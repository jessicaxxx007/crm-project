CRM客户关系管理系统
 
项目简介
本项目是面向中小型企业的CRM客户关系管理系统，基于B/S架构开发，围绕客户全生命周期进行管控，整合营销、客户、售后、权限、数据统计等能力，解决客户流失、销售流程不规范、人工统计效率低等问题。项目采用主流企业级开发方案，代码结构规范，适合Java Web学习与二次开发。
 
技术栈
 
后端
SpringBoot、SpringMVC、MyBatis、MyBatis Generator、PageHelper、AOP、MySQL 8、Maven、MD5加密
 
前端
Layui、jQuery、FreeMarker、ECharts、Cookie
 
主要功能模块
 
1. 基础模块：账号登录、密码修改、退出登录、记住我、登录拦截、全局异常处理
2. 营销管理：营销机会新增、编辑、查询、逻辑删除与状态流转
3. 客户管理：客户信息、联系人、订单、交往记录、流失客户管理
4. 服务管理：服务工单创建、分配、处理、反馈与归档
5. 系统管理：用户、角色、菜单、字典、日志管理（RBAC权限控制）
6. 数据报表：多维度客户数据统计分析
 
项目亮点
 
- 遵循MVC分层架构，代码解耦，易维护扩展
- 实现MD5密码加密、Cookie加密、登录拦截，保障系统安全
- 全业务采用逻辑删除，支持分页、批量操作
- 统一异常处理、标准接口返回格式，符合企业开发规范
